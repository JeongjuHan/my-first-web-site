웹(Web)은 인터넷을 통해 정보를 공유하고 표시하는 방식을 말합니다. 웹은 World Wide Web(WWW)라고도 불리며, 전 세계적으로 연결된 컴퓨터 네트워크를 통해 사용자들이 웹 사이트를 방문하고 웹 페이지를 열람할 수 있습니다. 웹은 다음과 같은 주요 구성 요소로 이루어져 있습니다: 웹 서버(Server): 웹 서버는 클라이언트로부터 HTTP 요청을 받아들이고, 해당 요청에 대한 응답으로 웹 페이지나 데이터를 제공합니다. 웹 서버는 일반적으로 웹 호스팅 서비스 제공 업체, 기업의 내부 서버, 클라우드 서비스 등에서 운영됩니다. 클라이언트(Client): 클라이언트는 웹 서버에 HTTP 요청을 보내고, 서버로부터 받은 응답을 해석하여 웹 페이지를 표시합니다. 웹 브라우저(Chrome, Firefox, Safari 등)는 가장 일반적인 웹 클라이언트입니다. 사용자는 웹 브라우저를 통해 웹 페이지를 열어 볼 수 있습니다. 웹 페이지(Web Page): 웹 페이지는 텍스트, 이미지, 비디오 등의 요소로 구성된 문서입니다. HTML(HyperText Markup Language)로 작성되며, CSS(Cascading Style Sheets)와 JavaScript 등의 기술을 사용하여 디자인과 동적인 기능을 추가할 수 있습니다. URL(Uniform Resource Locator): URL은 웹 페이지나 다른 웹 리소스의 고유한 주소를 나타냅니다. 예를 들어, "https://www.example.com"은 example.com 도메인의 홈페이지를 나타냅니다. 하이퍼링크(Hyperlink): 하이퍼링크는 웹 페이지 간의 연결을 제공하는 링크입니다. 사용자는 하이퍼링크를 클릭하여 다른 웹 페이지로 이동하거나, 동일한 페이지 내에서 특정 위치로 스크롤할 수 있습니다.
